# Windows ML Integration

Windows ML integration refers to deploying AI models through the Windows ML runtime while still benefiting from RTX acceleration where supported. It gives Windows-native applications a standardized inference path.

## Why It Matters

- Simplifies deployment into Windows applications
- Supports common model families without forcing each app to build its own runtime
- Complements TensorRT when teams need a more platform-native API surface
