# Amazon Route 53

Amazon Route 53 is AWS's managed DNS and traffic routing service. It connects domain management, health-aware routing, and private DNS patterns for AWS-hosted systems.

## Core Functions

- Domain registration and DNS management
- Health checks and failover routing
- Traffic policies for latency, geography, or availability
- Private DNS inside AWS networking environments

## Related Services

- Elastic Load Balancing - Route traffic toward healthy application endpoints
- [[aws-s3]] - Static website hosting behind custom domains
- [[aws-cloudfront]] - CDN routing and edge delivery
- [[aws-vpc]] - Private hosted zones for internal DNS
