# Amazon SageMaker

Amazon SageMaker is AWS's managed machine learning platform for building, training, deploying, and governing ML systems. In this vault it appears as AWS's main ML platform, complementary to [[aws-bedrock]] for managed foundation model access.

## Core Areas

- Model development and training workflows
- Managed deployment and inference endpoints
- MLOps and lifecycle management
- Studio-based development environments
- Governance and catalog integration

## Why It Matters

- Centralizes ML workflows that would otherwise be split across multiple AWS services
- Connects data, training, deployment, and governance inside one platform
- Serves as the more customizable ML option alongside Bedrock's managed FM access

## Related Services

- Athena - SQL queries on S3 data
- Redshift - Warehouse and analytics integration
- [[aws-bedrock]] - Managed foundation models and agents
- [[aws-s3]] - Data lake and storage foundation
