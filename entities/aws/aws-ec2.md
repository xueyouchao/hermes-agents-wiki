# Amazon EC2 (Elastic Compute Cloud)

Amazon EC2 provides resizable compute capacity in the cloud. It's essentially virtual servers in the AWS cloud.

## Key Features

- **Instance Types:** Various families optimized for different workloads:
  - General Purpose (t3, m5, m7i)
  - Compute Optimized (c5, c6i, c7i)
  - Memory Optimized (r5, r6i, x1e)
  - GPU Instances (p4, p5, g4, g5)
- **Scalability:** Auto Scaling Groups for elastic capacity
- **Pricing:** On-Demand, Reserved Instances, Savings Plans, Spot Instances
- **Storage:** EBS volumes (block) and instance store

## Use Cases

- Hosting web applications and APIs
- Batch processing
- Running enterprise applications
- Big data analytics
- Gaming servers
