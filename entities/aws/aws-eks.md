# Amazon EKS

Amazon EKS is AWS's managed Kubernetes service for running production Kubernetes workloads across cloud, hybrid, and edge environments.

## Core Value

- Managed Kubernetes control plane
- Native integration with AWS networking, identity, and storage
- Support for cloud-hosted, hybrid, and on-premises Kubernetes patterns

## Common Use Cases

- Platform engineering and internal developer platforms
- Distributed training and inference systems
- Batch, analytics, and data infrastructure
- Migration of existing Kubernetes workloads onto AWS

## Related Services

- [[aws-ecs]] - AWS-native non-Kubernetes container orchestration
- Fargate - Serverless runtime option for some EKS workloads
- [[aws-s3]] - Storage and data integration point
- EKS Anywhere - On-premises extension of the EKS model
