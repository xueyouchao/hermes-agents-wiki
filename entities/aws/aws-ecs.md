# Amazon ECS

Amazon ECS is AWS's managed container orchestration service. It is the AWS-native alternative to Kubernetes for running containers across EC2 or serverless container infrastructure.

## Core Value

- Managed scheduling and orchestration for containers
- Tight integration with AWS networking, IAM, and storage
- Multiple runtime models including EC2-backed and Fargate-backed deployments

## Common Use Cases

- Application modernization
- Batch and background processing
- Containerized AI inference and workflow execution
- Hybrid or edge-connected container deployments

## Related Services

- [[aws-eks]] - Managed Kubernetes alternative
- Fargate - Serverless compute option for ECS tasks
- [[aws-ec2]] - EC2-backed cluster capacity
- [[aws-s3]] - Shared data and artifact storage
