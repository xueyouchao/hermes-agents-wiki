# Amazon CloudFront

Amazon CloudFront is AWS's content delivery network for serving applications, APIs, media, and static assets with global edge distribution.

## Core Capabilities

- Global edge delivery for low-latency content distribution
- CDN caching in front of applications and storage origins
- Edge execution patterns through Lambda@Edge
- Built-in DDoS protection through AWS Shield Standard

## Common Use Cases

- Static web delivery
- API acceleration
- Video and media delivery
- Software and game distribution

## Related Services

- [[aws-s3]] - Common origin for static assets
- [[aws-lambda|Lambda]] - Edge logic via Lambda@Edge
- AWS Shield - DDoS protection integrated with CloudFront
- [[aws-route53|Route 53]] - DNS routing toward CloudFront
