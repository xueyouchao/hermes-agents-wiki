# Temporal Cloud

Temporal Cloud is the managed service offering from Temporal.io for deploying and operating Temporal clusters without managing infrastructure.

## Features

- Fully managed Temporal Service
- Automatic scaling and availability
- $1,000 free credits for new users

## Security

Temporal Cloud never sees user code — the platform only sees:
- Workflow type names
- Activity task queues
- Execution metadata
