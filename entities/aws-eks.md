# Amazon Elastic Kubernetes Service (EKS)   
Amazon EKS is a managed service for building, running, and scaling production-ready Kubernetes applications across cloud, on-premises, and edge environments.
    ## Key Features     - **EKS Auto Mode:** Full automation of Kubernetes cluster management (compute, storage, networking) with one click   - **Hybrid Deployment:** EKS Hybrid Nodes, Outposts, or EKS Anywhere   - **S3 Files Integration:** Thousands of compute resources connect to S3 simultaneously   ## Core Benefits     | Benefit | Description | |---------|-------------| | Operational Efficiency | Automates infrastructure management | | Cost & Performance | Automatic capacity planning and scaling | | Security & Reliability | Global infrastructure, native AWS security integration | | Standardization | Consistent tooling across deployment locations |   ## Primary Use Cases   - **Generative AI:** Distributed training and inference deployments   - **Internal Development Platforms:** Standardized environments for dev teams   - **Data Platforms:** Scalable batch processing and big data   - **Migration & Modernization:** Lift-and-shift, microservice scaling   ## Customer Success      | Customer | Achievement | |----------|--------------| | Sony Interactive Entertainment | 5x faster deployments, 60% lower costs | | Riot Games | $10M annual infrastructure savings | | Jaguar Land Rover | 95% faster pipeline build time | | Omi | 70% cost reduction for 3D rendering |   ## Related Services     - [[aws-ecs|ECS]] — AWS-native container orchestration   - [[aws-fargate|Fargate]] — Serverless for EKS   - [[aws-s3|S3]] — Data persistence for Kubernetes workloads   - [[aws-eks|EKS Anywhere]] — On-premises Kubernetes     ## References
