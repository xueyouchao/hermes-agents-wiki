---# AWS Lambda
AWS Lambda is a serverless compute service that lets you run code without provisioning or managing servers.
## Key Features
- **Pay-per-execution:** You pay only for compute time consumed- **Event-driven:** Triggers from 200+ AWS services and SaaS apps
- **Scalability:** Automatic scaling from zero to thousands of concurrent executions- **Languages:** Node.js, Python, Go, Java, C#, Ruby, PowerShell
- **Limits:** 15 min max execution time, 10GB memory, 1k concurrent executions (adjustable)## Use Cases- Processing S3 image uploads
- REST APIs via API Gateway
- Real-time file processing
- IoT data processing
