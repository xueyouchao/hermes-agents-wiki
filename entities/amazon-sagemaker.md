# Amazon SageMaker
    Amazon SageMaker is a comprehensive hub integrating AWS ML and analytics capabilities. It provides a unified environment for the entire data lifecycle—from ingestion to model deployment and governance.
    ## Core Components (Next Generation)     **SageMaker AI:** Complete suite for building, training, deploying ML and Foundation Models (FMs)   - HyperPod for high-performance cluster training   - JumpStart pre-built solutions   - MLOps tools for lifecycle operationalization   **SageMaker Unified Studio:** Single development environment with serverless notebooks and AI agents   **SageMaker Catalog:** Centralized data governance via Amazon DataZone   **Lakehouse Architecture:** Unifies S3 data lakes and Redshift warehouses with Apache Iceberg support   ## Key Features     ### Amazon Q Integration   - Natural language SQL query generation   - Build and run data pipeline jobs   - Create and deploy ML models     ### Unified Data Access   - Federated queries across sources without data movement   - Single copy analytics reducing storage redundancy     ### Enterprise Governance   - Fine-grained permissions across all tools   - Responsible AI: classification, toxicity detection, guardrails   ## Integrated Capabilities   - **Generative AI:** Build with Amazon Bedrock in SageMaker IDE   - **SQL Analytics:** Leverage Amazon Redshift   - **Data Processing:** Athena, EMR, AWS Glue    ## Customer Impact      | Customer | Impact | |----------|--------| | Toyota | Unified data across connected cars, sales, manufacturing | | Charter Communications | "Better developer experience, faster time to market" | | NatWest Group | 50% faster access to new tools | | Carrier | Streamlined lakehouse with strict security |   ## Recognition   - **Gartner Magic Quadrant (2025):** AWS named Leader in Cloud DBMS for 11th consecutive year   ## Related Services      - [[aws-athena|Athena]] — SQL queries on S3 data   - [[aws-redshift|Redshift]] — Data warehouse integration   - [[aws-bedrock|Amazon Bedrock]] — Foundation models   - [[aws-s3|S3]] — Data lake foundation     ## References
