## Glossary of Terms

### Key Concepts
- **Sparse Volumetric Grid**: 3D data structure with memory-efficient storage
- **Voxel**: 3D pixel in a volumetric grid
- **Tensor Core**: GPU hardware for matrix operations
- **Ray Marching**: Rendering technique for volumetric data

### Technical Terms
- **Jagged Tensors**: Variable-sized tensor batches
- **Hierarchical Grids**: Multi-resolution data structures
- **Differentiable Operations**: Computations with gradient support
